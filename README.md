# 🎵 Sur v2.0 - Quantum AI Music Engine 

**Sur** is the world's first language-agnostic synthesizer that translates the raw sentiment of lyrics into real-time musical scales (Ragas) and frequencies directly inside your browser. 

Unlike traditional AI wrappers that rely on heavy server-side APIs or pre-recorded audio tracks, **Sur uses pure mathematical wave synthesis (Web Audio API)** to generate zero-copyright melodies on the fly.

---

## ✨ Features that make Sur Unique

* **🌐 True Multilingual Core:** It doesn't matter if your lyrics are in Hindi, English, Punjabi, Spanish, or Bengali. The engine parses the linguistic matrix to detect the emotional core (Bhava).
* **🧠 Real-Time Sentiment Mapping:** Automatically maps the "mood" of your lyrics into classic musical foundations (e.g., Joyous Major Pentatonic/Bhairavi or Melancholic Shivaranjani).
* **💻 100% Client-Side Synthesis:** No API keys, no server costs, and no data tracking. The music is synthesized from scratch using your browser's audio nodes.
* **🎨 Cyberpunk UI with Live Visualizer:** Features a stunning futuristic neon branding dashboard with a real-time frequency canvas visualizer.
* **🆓 100% Royalty-Free:** Since the audio waves are generated purely from mathematics in real-time, the output is completely copyright-free.

## 🛠️ Tech Stack Used

* **Frontend Core:** Semantic HTML5 & Responsive CSS3
* **Design Paradigm:** Futuristic Cyberpunk Dark Theme with Custom SVG Logo Animation
* **Audio Engine:** Advanced Web Audio API (Oscillators, Gain Nodes, Analyser Nodes)

## 🚀 How to Run (Trial Setup)

1. Clone or download this repository.
2. Open the `index.html` file in any modern browser (Chrome, Edge, Safari, Firefox).
3. Paste your lyrics, hit **INITIALIZE AI MUSIC SYNTHESIS**, and watch/listen to the mathematics turn into music!
